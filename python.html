A Python Primer

The first thing you ought to do is to install the Anaconda Distribution, and bring up the Jupyter Notebook interface, as detailed on the instructions page. Here are two notebooks to help you get started.

The first notebook introduces some of the basic syntax, data types, control structures, and flow in the language through a series of examples. Download the notebook, run it cell by cell, change things, and play around!

The second notebook introduces Pandas, a python library for treating data like a spreadsheet, programatically. The data set used is one we scraped from the Goodreads web site. We walk through loading, manipulating, and saving data

.

The data file for this Notebook can be found here. Download it and save it somewhere on your system (easiest if you save it in the same folder you started the ipython notebook from).

The data is in Comma Separated Value (csv) form, which (as it says) represents items by rows, with data features separated by commas. The first line is usually taken to be the column heads. You can save data in csv format from Numbers and Excel, when playing with your own data.
